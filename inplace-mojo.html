native2ascii:inplace

Full name:

org.codehaus.mojo:native2ascii-maven-plugin:2.0.1:inplace

Description:

Converts files with characters in any supported character encoding to one with ASCII and/or Unicode escapes.

This mojo converts files inplace, it is not recommended to execute it on src subdirectories.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: process-resources.

Required Parameters

Name Type Since Description
dir File - Both source and target directory.
Default value is: ${native2ascii.dir}.

Optional Parameters

Name Type Since Description
encoding String - The native encoding the files are in.
Default value is: ${project.build.sourceEncoding}.
excludes String[] - Patterns of files that must be excluded.
includes String[] - Patterns of files to include. Default is "**\/*.properties".

Parameter Details

dir:

Both source and target directory.
  • Type: java.io.File
  • Required: Yes
  • Default: ${native2ascii.dir}

encoding:

The native encoding the files are in.
  • Type: java.lang.String
  • Required: No
  • Default: ${project.build.sourceEncoding}

excludes:

Patterns of files that must be excluded.
  • Type: java.lang.String[]
  • Required: No

includes:

Patterns of files to include. Default is "**\/*.properties".
  • Type: java.lang.String[]
  • Required: No
